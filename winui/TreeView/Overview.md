# WinUI TreeView Overview

The Syncfusion<sup>Âź</sup> [WinUI TreeView](https://www.syncfusion.com/winui-controls/treeview) is a data-oriented control that displays data in a hierarchical structure with expanding and collapsing nodes. It is commonly used to illustrate a folder structure or nested relationships in an application.

## Key features

* `Enhanced performance`: Optimized view reuse strategy and flat rendering architecture for enhanced performance.
* `Bound and unbound modes`: Supports binding hierarchical data or adding unbound tree nodes.
* `On-demand loading`: Supports loading the nodes on demand when the end user expands the node.
* `Selection`: Supports selecting the nodes with different selection modes and keyboard navigation.
* `Templating`: Provides complete UI customization using the template and template selectors.
* `Root lines`: Supports showing the lines between tree nodes.
