# Interactive Features in WinUI Chart

`SfChart` supports the following [interactive features](https://www.syncfusion.com/winui-controls/charts/interactive-features) to interact with the chart:

* Crosshair
* Tooltip
* Trackball
* Zoom and Pan
* Selection
